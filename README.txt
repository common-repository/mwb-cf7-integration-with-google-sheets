== Description ==

**WITH MWB CF7 INTEGRATION WITH GOOGLE SHEETS, MAKE YOUR DATA CLEAR, LIMPID AND SIMPLIFIED BY SENDING ALL YOUR FORM SUBMISSIONS AND ENTRIES TO GOOGLE SHEETS.**

**MWB CF7 Integration with Google Sheets**  integrates Google Sheets with Contact Form-7. This plugin sends all the data of Contact Form 7 (submissions/ entries) as per the fields of Google Sheets.

As the visitor provides its information on your site, upon the form submission the data will instantly be sent to Google Sheets. In this way, you can create clear and simplified data over Google Sheets.
 
The most significant factor is that you can manage your business more efficiently as you will have data to gather customer information. You can develop a strategy to attract more leads, convert them into customers, and fulfill their needs.

== WHAT IS INCLUDED IN FREE  CF7 INTEGRATION WITH GOOGLE SHEETS  PLUGIN == 

**1)** **Integrate CF7 with Google Sheets Account**
Enter the API credentials (Client ID, Client Secret, Authorized Redirect URI) to connect CF7 with your Google Sheets account. On the plugin’s dashboard, you can view the connection status. Plus, you can reauthorize and disconnect the integration any time. 


**2)** **Filter CF7 submissions As Per Feeds**
You can filter the Contact form 7 submissions based on user input using AND/OR logic. This logic will filter CF 7 forms submissions and send it to Google Sheets depending on user inputs.


**3)** **Detailed Log**
MWB CF7 Integration with Google Sheets plugin will provide a detailed log of each Contact Form 7 entry sent to Google Sheets. Integrate your Contact Form 7 with Google Sheets that comes with logging of all the API interaction with Google Sheets for better error handling.


**4)** **Delete Plugin Data On Uninstall**
You can delete a huge amount of data stored in the database when the plugin is uninstalled

**5)** **Associate Form Fields with Google Sheets Fields**
You can create CF7 forms and link form fields with Google Sheets fields. You can easily sync form submissions/entries data over the Google Sheets account.

**6)** **Clear And Download Logs**
You can delete the logs after 7 days. Additionally, you can download them too whenever needed.

**7)** **Receive Error Email Notification**
MWB CF 7 Integration With Google Sheets sends Email notifications if any input error occurs in the process of data, sending entries over Google Sheets. In this manner, you will be notified of any slight error.

== Frequently Asked Questions ==

= How will CF7 Integration with CF7 Integration with Google Sheets plugin help me? =

It sends all the data ( form submissions/ entries  ) over Google sheets as per the fields of Google sheets. This can help in creating simplified data over Google sheets.

= Can we schedule the deletion of logs in the CF7 with MWB CF7 Integration with Google Sheets Plugin? =

Yes, you can schedule this by visiting the Contact >> Google Sheet >> Setting >> Delete Logs After N Days, enter the number of days, and click on Save.

= Can we download the detailed log files through this MWB CF7 with MWB CF7 Integration with Google Sheets plugin? =

Yes, you can download the detailed log files, the log folder will be created in wp-content/mwb-cf7-logs and date-wise logs of all the interaction with Google sheets will be stored here.

= Can we also delete the CF7 Integration with Google Sheet Plugin data? = 
Yes, you can delete the plugin data.

= Is there any limit to delete the logs and after how many days? =
You can set your limits as per your need and the minimum number of days is 7


= Can we delete the CF7 integration data with google sheets? =
Yes, you can delete the plugin data. To enable this feature, go to Setting, click on the Plugin Data toggle, and click on Save.
